Sobre Mim

Desde cedo, sempre fui movida pela curiosidade de entender como as coisas funcionam. Quando eu tinha 12 anos, passava horas personalizando meu Tumblr, testando cores, fontes e layouts. O que começou como um simples hobby logo se transformou em curiosidade: como tudo aquilo funcionava por trás da tela? Queria entender não só a aparência, mas a lógica por trás de cada interação. Foi essa inquietação que me levou a escolher Análise e Desenvolvimento de Sistemas, onde finalmente mergulhei de vez no universo do desenvolvimento.

Volocopter

Participei de um projeto acadêmico onde desenvolvemos um website completo para a empresa Volocopter, com foco na promoção de seus produtos e serviços alinhados aos Objetivos de Desenvolvimento Sustentável (ODS). Fiquei responsável pela criação e implementação da interface da página de Responsabilidades, desde o protótipo visual no Figma até o desenvolvimento com HTML, CSS e JavaScript. Trabalhei em equipe para garantir a consistência do site, adaptabilidade para diferentes dispositivos e navegadores, além de aprender novas tecnologias para atender aos requisitos do projeto.

Segurize

Participei do desenvolvimento de uma API REST para o gerenciamento de seguros, utilizando Java com Spring Boot e Spring Security, focando na segurança e conformidade com a LGPD. Fiquei responsável pelo desenvolvimento do frontend, criando a interface e design com React e JavaScript, além de contribuir para a implementação do backend. Trabalhei na criação de CRUDs, consumo de APIs com Fetch API e Axios, e na integração com bancos de dados MySQL e PostgreSQL. O projeto seguiu metodologias ágeis e foi hospedado na plataforma Render, garantindo eficiência e escalabilidade.
